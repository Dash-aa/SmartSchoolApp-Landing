# SmartSchoolApp-Landing

Лендінг бакалаврської роботи, присвяченої розробці вебзастосунку Smart School для автоматизації навчального процесу.

## Мета
Створити односторінковий вебсайт для представлення бакалаврської роботи з урахуванням вимог SEO, доступності, адаптивності та правильної організації Git-історії.

## Технології
- HTML5
- CSS3
- Git
- GitHub Pages
